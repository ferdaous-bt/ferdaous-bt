### 🌟 Featured Projects

#### 💊 [Drug Distribution Risk Prediction](https://github.com/ferdaous-bt/risque-distribution-pharma-tn)
End-to-end binary classification model predicting pharmacy distribution for **6,053 Tunisian drugs**.
- 🎯 **F1 = 0.81 | AUC = 0.891** with XGBoost + SMOTE
- 🔍 Web scraping → EDA → Feature engineering → SHAP interpretability → Streamlit deployment
- **Stack:** Python · XGBoost · Scikit-learn · SHAP · Streamlit

#### 🧠 [Deep Learning Emotion Classification](https://github.com/ferdaous-bt/deep-learning-emotion-classification)
Neural network-based emotion classification on unstructured data.
- 🧪 Exploration of deep learning architectures for classification tasks
- **Stack:** Python · Deep Learning · Jupyter

#### 🩺 [Diabetes Prediction with Machine Learning](https://github.com/ferdaous-bt/diabetes-prediction-ml)
Binary classification on the Pima Indians dataset (768 patients) with full preprocessing pipeline.
- 🎯 **76% accuracy, 74% recall** with Random Forest
- 📉 **42% reduction** in false negatives vs. baseline
- **Stack:** Python · Scikit-learn · Pandas · NumPy

#### 🛍️ [E-Commerce Data Warehouse & Business Intelligence](https://github.com/ferdaous-bt/ecommerce-data-warehouse)
Star schema Data Warehouse with a robust ETL pipeline processing **541,000 transactions**.
- ✅ **97.7% retention rate** in the ETL pipeline
- 📊 6 BI dashboards revealing 35% seasonal growth & 65% UK market concentration
- **Stack:** Python · PygramETL · Matplotlib
